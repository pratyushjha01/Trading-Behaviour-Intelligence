# 📈 Trading Behaviour Intelligence

> **Understanding how trading behaviour changes across different market phases through data-driven analysis.**

---

## 📌 Project Overview

Most trading dashboards focus on **profit, win rate, or returns**. While these metrics explain *what* happened, they rarely explain **why different trading outcomes occur across changing market conditions.**

This project was built with a different objective.

Instead of analysing a single trading account, I analysed **four independent account journeys**, each representing a different phase of trading performance.

Rather than asking,

> *"Which account made the most profit?"*

this project explores,

> *"How does trader behaviour change during profitable phases, losing phases, recovery periods, and capital protection?"*

The goal was to transform raw trading records into a behavioural analytics dashboard that explains **how different trading phases influence long-term profitability.**

---

# 💡 Why I Built This Project

While reviewing my own trading history, I noticed that every account told a different story.

Some accounts experienced rapid growth within only a few profitable sessions.

Some suffered deep drawdowns before recovering.

Others remained profitable despite extended losing streaks.

This raised an important question:

> **Can trading performance be understood better by analysing behaviour across different market phases rather than looking only at final profit?**

That question became the foundation of this project.

Instead of treating each account as separate performance reports, I transformed them into four behavioural case studies representing different stages of a trader's journey.

---

# 🎯 Project Objective

The objective of this project is to analyse trading behaviour across multiple account journeys and identify how different decisions influence long-term performance.

The analysis focuses on answering questions such as:

- How quickly can an account grow during profitable phases?
- How does disciplined execution outperform rapid gains over time?
- How can traders recover from prolonged drawdowns?
- How can profits be protected during difficult market conditions?
- Which behavioural patterns contribute to long-term consistency?

---

# 📂 Dataset

The dashboard is built using **195 real trading records** collected from **four independent XAUUSD (Gold) trading accounts.**

### Data includes:

- Trade Date & Time
- Account Name
- Trade Direction
- Position Size (Lots)
- Net Profit / Loss
- Winning & Losing Trades
- Gross Profit & Gross Loss

Each account represents a different behavioural journey instead of a different trading strategy.

---

# 📊 Dashboard Overview

The dashboard is divided into three analytical sections.

---

## 📄 Page 1 — Performance Overview

This page provides a high-level summary of overall trading performance.

It introduces the key business metrics and establishes the context for deeper behavioural analysis.

### Key KPIs

- Net Profit
- Total Trades
- Win Rate
- Profit Factor
- Total Position Size
- Average Win
- Average Loss

### Key Visuals

- Net Profit by Account
- Profit Factor by Account
- Win/Loss Distribution

This page answers one simple question:

> **"What happened across all trading accounts?"**

---

## 📄 Page 2 — Behaviour Analysis

Instead of focusing only on profitability, this section explains **why** certain accounts remained profitable.

The analysis demonstrates that:

- Larger winning trades compensated for more frequent losing trades.
- Controlled losses created a favourable risk-reward profile.
- Position sizing adapted to market conditions instead of emotions.

Rather than analysing trading results alone, this page evaluates the behavioural decisions behind those results.

---

## 📄 Page 3 — Performance Through Phases

The final section visualises how each account evolved over time using cumulative equity curves.

Each account represents a unique trading phase:

### 🟢 Steady Growth

Consistent execution produced the highest final profitability, demonstrating that disciplined growth can outperform fast starts.

### 🟣 Fast Profit Phase

Shows how capital can compound rapidly during favourable market conditions without relying on larger position sizes.

### 🔵 Drawdown Recovery

Illustrates how disciplined execution can recover substantial losses and restore profitability over time.

### 🟡 Profit Protection

Demonstrates how protecting earlier gains during prolonged losing periods is equally important as generating profits.
